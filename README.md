# ☕ Cheza Kama Wewe

Cheza Kama Wewe is a Kenyan creator support platform that allows fans to financially support their favorite content creators.

## 🚀 Features (MVP)

- Creator profiles
- 5% platform fee per transaction
- Creator dashboard
- Manual withdrawal system
- Built with React + Vite + Tailwind

## 🌍 Vision

To empower Kenyan creators by giving them a simple way to receive direct support from fans.

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase (coming soon)
- Vercel Deployment

## 🔥 Status

MVP in development.
